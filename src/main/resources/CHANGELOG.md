# Changelog

## [1.0] - 2025-12-14
### Изменения
- Очереди и игры переписаны. Теперь есть разные виды очередей и игр:
  - НОРМАЛ - до 30-ти человек, можно присоединяться во время игры.
  - РАНГ - ровно 10 человек, нельзя присоединяться во время игры.
  - КАСТОМ - создаются игроками, игру можно настраивать.
- Точки теперь нужно захватывать по порядку.
- Грязевые бомбы (яйца) теперь не должны пролетать сквозь игроков.
- Вся система дропа предметов переписана на более простую. Теперь предметы должны быть более рандомномными.
- Теперь у каждого иргока есть 3 основные статистики (они отображаются в табе):
  - Уровень - повышается с опытом за каждую игру (не кастомную).
  - Ранг - повышается или понидается в ранговых играх.
  - Скилл - соотношение убийств к смертям.

## [1.1] - 2025-12-18
### Изменения
- Нормал игры теперь в режиме захвата точек не по порядку. Можно захватывать любые точки.
- В боковой панели указана ваша команда.
- Тиммейты теперь подсвечиваются.
- Ранг и Скилл убраны из таба чтобы не занимать места. Остался только уровень.

## [1.2] - 2025-12-21
### Изменения
- Очки теперь падают медленнее. За каждые две точки разницы -1 очко. (При всех 5-ти захваченных точках потеря у врага будет -3 в секунду, вместо -5)
- Миры теперь чередуются чтобы в каждой игре был новый биом.

## [1.3] - 2025-12-22
### Изменения
- Карты в обычных и ранговаых играх теперь выбираются рандомно.
- У щитов теперь ограниченная прочность, они ломаются.

## [1.4] - 2025-12-27
### Изменения
- Длительность поглощения золотого нагрудника уменьшена: 5 -> 3 сек.
- Танковый нагрудник испрвлен. Теперь он наносит правильный урон.
- Все блоки теперь цвета команды.
